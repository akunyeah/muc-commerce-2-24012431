## 用自己的话回答

- 什么是特征，什么是标签：特征是用于预测的数据，标签是预测的结果数据，在上述情景中可以理解为分析字段和结果字段。
- 为什么要保留测试集：如果测试集与训练集相同，那么模型的准确率会过高，出现过拟合现象，导致模型在真实数据上的表现不佳。
- 为什么83%准确率仍可能没有用：虽然准确率高，但是其召回率为0，不具有实际意义。
